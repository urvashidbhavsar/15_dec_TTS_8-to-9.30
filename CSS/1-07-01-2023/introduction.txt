there is a three ways to create(prepare) css:
(1) inline css -> apply in the Tag.
(2) internal css -> using <style> tag in <head> section.
(3) External css -> create an external file and link with <link> tag.

css syntax : (internal and external)
selector {
    properties : value;
}

inline css :
style = " properties : value "
